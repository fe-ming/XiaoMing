# XiaoMing
### 开发环境
- VS2015或者VS2017(win10)
- .NET Framework 4.0
- CefSharp.WinForms 49.0.0
- 目标平台x86或者x64
### 开发注意事项
- clone下代码后，第一次打开项目需要重新生成一次
- 调式或运行时Any CPU需要改为x86或者x64
### 目前已完成的功能
- Tab页签的添加和删除
- 切换内核模式（ie和chrome）
- 前进、后退和刷新功能
- Tab页签动态显示网页标题、地址栏显示当前网页URL地址
